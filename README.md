# UniBooktime-
Uni Booktime productivity and study app
- Added starter code skeleton (UniBooktime.json) for app structure
- Added Google Sheet template with sheets:
  - Users: name, country, phone, email, profile picture
  - Assignments: title, description, deadline, status
  - StudyMaterials: file name, file URL, file type
  - DailyGoals: date, goal description, completed, time spent
  - TimerSettings: default time, max time, break time
  - Languages: language name, code
- Added placeholder for AI Bot integration
- Added placeholders for QR code scanner and generator
- Added placeholders for Focus Timer, Meditation break, Notifications, and Help button
- Prepared for multi-language support and profile features
